# AI Toolbox

A collection of utilities and scripts for working with AI coding assistants.

## Tools

### [claude-code-statusline](./claude-code-statusline/)

A lightweight status line script for [Claude Code](https://docs.anthropic.com/en/docs/claude-code) that shows real-time context window usage, session cost, duration, compaction count, and working file count.

```
 project | main | Opus 4.6 | ████████░░░░░░░░░░░░ 42% 65.5k/156.0k | $0.34 | 12m | 0x compact | 5 files | a1b2c3d4
```
